safe-stringify-ts

A TypeScript utility for safely stringifying any JavaScript object

Features

Basic Usage

Stringify objects with proper formatting and type handling

import { safeStringify } from 'safe-stringify-ts';

const obj = {
  string: 'Hello World',
  number: 42,
  boolean: true,
  date: new Date(),
  nested: { a: 1, b: 2 }
};

console.log(safeStringify(obj));

Circular References

Safely handle objects with circular references

import { safeStringify } from 'safe-stringify-ts';

const obj = { nested: { data: 'value' } };
obj.self = obj;

console.log(safeStringify(obj));

DOM Node Handling

Stringify DOM nodes and browser objects

import { logToElement } from 'safe-stringify-ts';

const data = { 
  element: document.body,
  windowRef: window,
  otherData: [1, 2, 3]
};

logToElement(data, outputElement);

Async Logging

Log results of async operations

import { logAsync } from 'safe-stringify-ts';

async function fetchData() {
  const response = await fetch('https://jsonplaceholder.typicode.com/todos/1');
  return response.json();
}

logAsync(fetchData());

Installation

npm install safe-stringify-ts

API Documentation

safeStringify(value, options)

Safely converts any JavaScript value to a JSON string, handling circular references and special objects.

Options:

  • maxDepth: Maximum depth to traverse objects (default: 10)
  • indentation: Space value for JSON.stringify (default: 2)
  • showFunctions: Include function names and bodies (default: true)
  • dateFormat: Format for dates ('iso', 'locale', or 'timestamp')

logToElement(value, element, label)

Logs a formatted representation of a value to an HTML element with syntax highlighting.

logAsync(promise, label, options)

Logs the result of a Promise when it resolves or the error if it rejects.

createLogger(options)

Creates a configurable logger instance with various formatting options.
